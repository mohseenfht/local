Local Plugin - Dynamic Pages
======================

Information
-----------

This plugin allow add new menu items at the end of Navigation Menu in order to customize your options menu.

Version  
-------
Moodle 2.7+ and 2.6+

Download way
- Download the zip 
- Unzip it into  local/ folder in your Moodle,
- Rename the new folder "moodle-local_pagedynamic" to "navigation"
- Enjoy!!!
